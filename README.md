# ❤️ Heart Disease Prediction (Machine Learning)

## 📌 Problem Statement

The objective of this project is to predict the likelihood of heart disease based on patient health data using machine learning techniques.

---

## 📊 Dataset

* Contains 300+ patient records
* Features include age, cholesterol, blood pressure, heart rate, etc.

---

## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

---

## 🔍 Process

* Data cleaning and preprocessing
* Feature scaling and train-test split
* Model building (Logistic Regression, Random Forest)
* Model evaluation using confusion matrix and classification metrics

---

## 📈 Results

* Achieved ~78–82% accuracy
* Evaluated using precision, recall, and F1-score

---

## 🔑 Key Insights

* Age, cholesterol, and maximum heart rate significantly influence heart disease risk
* Machine learning models can effectively predict disease probability

---

## 🚀 Conclusion

This project demonstrates how machine learning can be used in healthcare to assist in early detection and risk prediction of heart disease.
